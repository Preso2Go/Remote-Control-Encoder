This application should be copied to a location that the application is able to wite something to.
This application will create a small file with the encrypted login credentials. 
This way, the user only has to login once and the next time you start the application it will automaticly login for you.

If you want to change the login credentials, please delete the .config file inside the folder of the .exe file wich has the same name.
The next time you start the application, you will see the login screen.

For more information, please visit: http://www.presentations2go.eu

Please note: this application is meant to be an example of how to use the RemoteController API in C#.
For more information on the API, please visit: http://upgrade.presentations2go.eu/p2g.api/Encoder.RemoteControl/APIv3/
